URL Shortener

  • Generating short URLs:
  • Pass a URL as a parameter to /new/:URL
  • Properly formatted URLs will return a new short URL
  • URLs that are not properly formatted will return an error
  • Example of a properly formatted URL: https://www.example.com
  • Example of an improperly formatted URL: www.example.com
  • Using short URLs:
  • Navigating to a valid short URL will redirect you to the full URL
  • Entering an unrecognized short URL will return an error
